## 🗂️ 変更が必要なファイルの場所

料金を変更するには、以下の2つのファイルを編集する必要があります：

```
tiger-dojo/
├── apps/frontend/app/page.tsx                    # トップページの料金表示
└── apps/frontend/components/subscription/PricingPlans.tsx  # 料金プランコンポーネント
```

## 💰 料金表示の変更手順

### 1. トップページの料金表示を変更

**ファイル位置**: `apps/frontend/app/page.tsx`

このファイルを開き、以下の部分を探してください：

```tsx
// 現在の設定（例）
<div className="text-4xl font-bold text-white mb-2">¥15,000</div>
<div className="text-gray-300 text-sm">月額</div>
```

**変更方法**:
- `¥15,000` の部分を新しい金額に変更
- 例：`¥20,000` に変更したい場合は `¥20,000` と書き換える

### 2. 料金プランコンポーネントの変更

**ファイル位置**: `apps/frontend/components/subscription/PricingPlans.tsx`

このファイルを開き、以下の部分を探してください：

```tsx
// スタンダードプランの料金
const plans = [
  {
    name: 'スタンダードプラン',
    price: '¥15,000',  // ← ここを変更
    originalPrice: '¥25,000',  // ← 元値も変更可能
    // ... 他の設定
  },
  {
    name: 'プレミアムプラン', 
    price: '¥40,000',  // ← ここを変更
    originalPrice: '¥60,000',  // ← 元値も変更可能
    // ... 他の設定
  }
]
```

**変更方法**:
1. `price:` の後の金額を新しい料金に変更
2. `originalPrice:` の後の金額も必要に応じて変更（元値・定価）

## 🔗 PayPalリンクの変更手順

料金を変更した場合、PayPalの決済リンクも変更する必要があります。

### PayPalリンクの場所

以下のファイルでPayPalリンクを管理しています：

1. **CLAUDEファイル**: `CLAUDE.md`
2. **料金プランコンポーネント**: `apps/frontend/components/subscription/PricingPlans.tsx`

### 1. CLAUDEファイルのリンク更新

**ファイル位置**: `CLAUDE.md`

以下の部分を探してください：

```markdown
## 💳 PayPal決済リンク
- **スタンダードプラン (¥15,000)**: https://www.paypal.com/webapps/billing/plans/subscribe?plan_id=P-8PD885481X888505ENAAKQAQ
- **プレミアムプラン (¥40,000)**: https://www.paypal.com/webapps/billing/plans/subscribe?plan_id=P-6NX871965X4205206NAAKOJI
```

**変更方法**:
1. 金額部分（`¥15,000`, `¥40,000`）を新しい料金に変更
2. PayPalで新しい料金プランを作成し、新しい`plan_id`を取得
3. URLの`plan_id=`以降の部分を新しいIDに変更

### 2. 料金プランコンポーネントのリンク更新

**ファイル位置**: `apps/frontend/components/subscription/PricingPlans.tsx`

以下の部分を探してください：

```tsx
const plans = [
  {
    name: 'スタンダードプラン',
    // ... 他の設定
    paypalLink: 'https://www.paypal.com/webapps/billing/plans/subscribe?plan_id=P-8PD885481X888505ENAAKQAQ'
  },
  {
    name: 'プレミアムプラン',
    // ... 他の設定  
    paypalLink: 'https://www.paypal.com/webapps/billing/plans/subscribe?plan_id=P-6NX871965X4205206NAAKOJI'
  }
]
```

**変更方法**:
- `paypalLink:` の後のURLを新しいPayPalプランのURLに変更

### PayPal側での新しいプラン作成方法

1. **PayPal管理画面にログイン**
   - PayPal Business アカウントでログイン
   - https://www.paypal.com/businessmanage/account/

2. **定期支払いプランを作成**
   - 「商品とサービス」→「定期支払い」に移動
   - 「プランを作成」をクリック
   - プラン名：例「FXトレード道場スタンダードプラン」
   - 料金：新しい金額を設定（例：¥18,000）
   - 請求サイクル：月次

3. **プラン作成後にリンクを取得**
   - 作成したプランの「共有」または「リンクを取得」をクリック
   - 表示される URL をコピー
   - 例：`https://www.paypal.com/webapps/billing/plans/subscribe?plan_id=P-新しいID`

4. **古いリンクを新しいリンクに置き換え**
   - 上記のファイルで古いPayPalリンクを新しいものに変更

## ✅ 変更後の確認方法

### 1. 表示の確認
1. 開発サーバーを起動：`npm run dev`
2. ブラウザで http://localhost:3000 にアクセス
3. トップページで料金が正しく表示されているか確認
4. 料金プランページで両方のプランの料金が正しいか確認

### 2. PayPalリンクの確認
1. 各プランの「今すぐ申し込む」ボタンをクリック
2. PayPalの正しいプランページに移動するか確認
3. PayPal側で表示される金額が正しいか確認

## ⚠️ 注意事項

### 重要な注意点
1. **ファイルのバックアップ**: 変更前に必ずファイルのコピーを作成
2. **PayPal設定**: 古いPayPalプランは無効化し、新しいプランのみ有効にする
3. **テスト**: 変更後は必ずテスト購入で動作確認を行う
4. **既存顧客**: 既存の顧客の契約には影響しないことを確認

### ファイル編集時の注意
- **文字コード**: ファイルはUTF-8で保存
- **インデント**: 既存のインデント（空白やタブ）を崩さない
- **記号**: `"` や `'` などの記号を間違えないよう注意
- **コンマ**: 項目の後の `,` を忘れない

### エラーが出た場合
1. ブラウザのコンソール（F12キー）でエラーメッセージを確認
2. ファイルの構文エラーがないかチェック
3. 変更前の状態に戻して再度確認
